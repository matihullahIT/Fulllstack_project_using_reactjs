# Site 2

A React.js project.

## Getting Started

1. **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd Site-2
    ```

2. **Install dependencies:**
    ```bash
    npm install
    ```

3. **Start the development server:**
    ```bash
    npm start
    ```

## Features

- Modern React.js setup
- Easy to customize
- Fast development workflow
